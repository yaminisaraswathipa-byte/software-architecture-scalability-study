# Software Architecture Scalability Study

This repository contains raw metrics generated using the **Radon Python analysis tool** for the research project.

Metrics generated:
- Lines of Code (LOC)
- Cyclomatic Complexity
- Maintainability Index

Files:
- `example.py` – Python file used for analysis
- `raw_metrics.txt` – LOC metrics generated using `radon raw`
- `complexity_metrics.txt` – Cyclomatic complexity metrics generated using `radon cc`
- `maintainability_metrics.txt` – Maintainability index generated using `radon mi`
